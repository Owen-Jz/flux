Iconography

Guidelines for icon usage, sizing, and custom creation

Icon Libraries

Primary Library
Lucide React

Import from lucide-react. Our go-to for all UI icons. Consistent 1.5px stroke weight, rounded caps, 24x24 viewBox.

Secondary Library
Heroicons

Import from @heroicons/react. Use for specialized illustrations or when Lucide lacks coverage.

Icon Sizing Rules

Context Size Usage
Inline 16px Text paragraphs, badge labels, form elements
Buttons 20px Icon buttons, action buttons, dropdown triggers
Navigation 24px Sidebar items, tab icons, breadcrumb icons
Feature Icons 32px Empty states, onboarding illustrations, feature highlights

Icon Color Rules

Default to currentColor to inherit from parent text color. Use --flux-brand-primary for brand moments only.

currentColor
#7E3BE9

Custom Icon Creation Guidelines

When creating custom icons, follow these specifications to maintain visual consistency:

  • ViewBox: 24x24
  • Stroke width: 1.5px consistent
  • Stroke linecap: round
  • Stroke linejoin: round
  • Corner radius: Match Lucide style
  • Optical balance: Center visually, not mathematically

Do's and Don'ts

Do's
  • Use icons consistently from the same library
  • Match icon weight (1.5px stroke) across all elements
  • Keep icons optically centered within their viewBox
  • Use icons at specified sizes for their context
  • Apply currentColor for flexible theming
Don'ts
  • Mix icons from different libraries with mismatched styles
  • Use icons smaller than 16px or larger than 32px
  • Stretch or distort icons away from their aspect ratio
  • Apply hard-coded colors when currentColor would work
  • Stack multiple icons in the same interactive element

Imagery & Photography

Standards for photography, illustration, and image treatment

Photo Style Guide

Real Teams
Document authentic collaboration and workflows
Diverse & Inclusive
Represent varied backgrounds, roles, and perspectives
Natural Lighting
Prefer soft, diffused light with warm undertones

Slightly desaturate or color-grade photos to harmonize with the brand palette. Avoid overly bright or high-contrast imagery that clashes with the purple accent system.

Image Treatment

Border Radius
--flux-radius-lg = 12px

Apply consistent rounded corners to all photographs and featured images.

Shadow
--flux-shadow-md

Use subtle drop shadows to lift images from the page. Avoid hard shadows.

Illustration Style

If illustrations are used, keep them minimal line art with brand purple (#7E3BE9) accents. Avoid complex gradients or heavy fills.

What to Avoid

  • Stock photo clichés: staged handshakes, fake phone calls, artificial smiles
  • Overly staged corporate photography with artificial lighting
  • Outdated tech imagery: legacy software, old devices, deprecated UI patterns

Image Format Guidance

Preferred
WebP

Best compression and quality. Use for all photography and illustrations.

Fallback
JPEG

For photographic content where WebP lacks support.

Transparency
PNG

Only when transparency is required. Avoid for photos.
